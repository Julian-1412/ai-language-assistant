# AI Language Assistant

A RAG-based AI Language Assistant designed to handle incoming requests via webhooks, ingest data, and leverage LLMs for intelligent responses.

## Key Features
- **RAG (Retrieval-Augmented Generation)**: Enhanced responses using local document knowledge.
- **Webhook Integration**: Real-time communication for messaging platforms.
- **n8n Integration**: Workflow automation ready.
- **Escalation Logic**: Seamless transition to human support when needed.

## Setup Instructions
1. Install dependencies: `npm install`
2. Configure environment: `cp .env.example .env`
3. Start the application: `npm start`
